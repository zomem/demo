## 文章小程序全栈开发  

#### 技术框架   
|类型|技术|
| -- | --|
|后台|express|
|小程序|微信原生开发|
|数据库|Mysql和MongoDB|
|数据库连接|[access-db](https://github.com/zomem/access-db)|
|服务器|阿里云轻量服务器centos7|
|文件加速|阿里云cdn|

![图片1](./d1.png)![图片2](./d2.png)  
  
  
#### 使用  

**数据库**
将`bidu.sql`导入到mysql数据库。并在`.env`里配置mysql用户和密码  
  
**服务器**  
  
```bash
cd server
npm install
npm run start
```
  
**小程序**  
  
用微信开发工具打开`mini`即可。
